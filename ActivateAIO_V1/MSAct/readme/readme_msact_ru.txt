 			    Назначение программы
			————————————————————————————
Программа предназначена для установки ключей и активации Windows 7-10 и Office 2010-2013.
Если установленный ключ допускает активацию по сети, через интернет, продукт активируется
автоматически, если этого не происходит и ключ не заблокирован - выдаётся код для передачи
в Центр Активации M$ для активации по телефону.
В отличии от MSAct в программе MSAct Plus имеется возможность создавать группы ключей для 
разных продуктов. 

 			    Работа с программой
			————————————————————————————
1. Выберите продукт для установки ключа и активации;
2. Введите в поле для ввода ключей ключ для выбранного продукта;
3. Нажмите кнопку "Установить ключ".

Если установленный ключ допускает активацию по сети, через интернет, продукт активируется
автоматически, иначе, если ключ не заблокирован, в полях "1-9" отобразится код для передачи 
в Центр Активации M$ для активации по телефону.
Для активации по телефону необходимо позвонить в Центр Активации, передать код из полей "1-9"
и получить ответный код, который необходимо ввести в поля "A-H" и нажать кнопку "Применить".
Если всё выполнено без ошибок продукт активируется.
Когда активация выполнена можно сохранить ключ и ответный код в текстовый файл с помощью 
специальной кнопки. Впоследствии, например после переустановки системы, достаточно
будет установить ключ, ввести в программу сохранённый ранее ответный код и нажать кнопку "Применить".
Восстановление активации возможно даже если ключ уже будет заблокирован.

*** Если скопировать в буфер обмена сохранённый ответный код и установить курсор на поле "A", код
будет вставлен автоматически.
*** Кнопочка возле блока "9" принудительно отображает код предназначенный для передачи в Центр 
Активации, даже если ключ заблокирован. Даёт возможность выполнить активацию по телефону для
ключей проходящих активацию онлайн.

			   Работа с Базой Ключей
			————————————————————————————
В окне Базы Ключей можно создавать свои группы ключей, удалять, добавлять в группы ключи из буфера
обмена, загружать группы в поле для ключей. После загрузки группы становится доступным автоматический
режим работы программы. В этом режиме обрабатывается весь список ключей до тех пор пока продукт не 
активируется или не закончится список.
